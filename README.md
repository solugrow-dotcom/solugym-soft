### 1. Install Dependencies
```bash
npm install
```

### 2. Configure Environment
A `.env.local` file has been created for you.
- **Demo Mode**: Works out of the box (uses mock data).
- **Live Mode**: Add your keys to `.env.local`:
  ```env
  NEXT_PUBLIC_INSFORGE_API_URL=https://your-project.supabase.co
  NEXT_PUBLIC_INSFORGE_API_KEY=your-anon-key
  ```

### 3. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
